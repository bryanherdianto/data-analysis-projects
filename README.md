# Data Analysis Projects

A collection of data analysis projects covering exploratory data analysis (EDA), SQL, and Python programming. Some of these projects are from DataLemur, DataCamp, and freeCodeCamp.

## Repository structure

```
data-analysis-projects/
├── airlines/                                  # EDA projects (dataset + notebook.ipynb)
├── brazil-forest-fire/
├── cars-dataset/
├── covid19/
├── flood-analysis/
├── gender-analysis/
├── london-housing/
├── nobel-winners/
├── police-dataset/
├── ride-sharing/
├── titanic/
├── udemy-courses/
├── weather/
├── demographic-data-analyzer/                 # freeCodeCamp certification projects
├── mean-variance-standard-deviation-calculator/
├── medical-data-visualizer/
├── page-view-time-series-calculator/
├── sea-level-predictor/
└── sql/
    ├── datacamp-sql/                          # guided SQL analysis notebooks
    ├── datalemur-sql/                         # standalone SQL interview solutions
    └── murder-mystery-game/                   # SQL murder mystery puzzle
```
